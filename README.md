Installation

Clone the repo and cd into it

composer install
npm install
npm run dev
Rename or copy .env.example file to .env
Set your IGDB_KEY in your .env file. You can get an API key here.
php artisan key:generate
php artisan serve
Visit localhost:8000 in your browser
